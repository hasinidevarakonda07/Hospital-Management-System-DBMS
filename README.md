# Hospital-Management-System-DBMS
A database-driven Hospital Management System that streamlines patient, doctor, appointment, and billing records. Developed using DBMS concepts including ER diagrams, normalization, schema design, and SQL queries for efficient data management.


## Overview
A database management system designed to manage hospital operations including patient records, doctor information, appointments, and billing.

## Features
- Patient registration
- Doctor management
- Appointment scheduling
- Billing records
- Database querying and reporting

## Database Design
The system uses a relational database model with normalized tables to ensure data consistency.

## Technologies
- DBMS
- SQL
- ER Modeling
- Database Normalization

## Project Components
- ER Diagram
- Schema Design
- SQL Queries
- Project Documentation
